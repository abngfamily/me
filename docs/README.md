# Nikhil George — Personal Website

## What this is

A warm, professional personal website for Nikhil George. It showcases his professional background, career experience, and personal interests.

## Tech stack

* Plain HTML + CSS (no frameworks needed — fast and simple)
* Hosted with Python's built-in HTTP server
* Google Fonts: Playfair Display (headings) + Inter (body)

## How to run

```py
python3 -m http.server 3000
```

Then open your browser at http://localhost:3000

## Pages / Sections

1. **Hero** — Name, photo, tagline, short summary
2. **About Me** — Who Nikhil is in his own words
3. **Professional Expertise** — Two cards: Product/Technology Leadership & Organisation Leadership
4. **Experience** — Full career timeline from CMU → Microsoft → Mobisante → VW → Logitech → Precision Neuroscience → Meta
5. **Beyond Work** — Three interest cards: Programming, Guitar, Watercolor Painting
6. **Contact / Footer** — LinkedIn link

## Key files

* `index.html` — All the content and page structure
* `static/style.css` — All the colours, fonts and layout
* `uploaded/Me.jepg` — Nikhil's profile photo (referenced directly from uploads)
